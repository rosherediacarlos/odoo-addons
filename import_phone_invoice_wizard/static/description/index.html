Atzeneta - Importar Facturas

Este modulo crea un nuevo elemento de menu "Facturacion / Contabilidad" --> "Clientes" --> "Importar Facturas".
El cual mostrara un wizard para insertar fecha de la factura y el fichero csv que vamos a leer y procesar.
A continuacion mostramos un enlace para ver la cabecera del fichero csv, esta debe ser siempre la misma, aun que podemos cambiar el ordern. Ejemplo fichero csv

Instalacion

Este necesita las siguientes dependencias:

  1. account
  2. sale
  3. purchase
  4. account_invoice_force_number
